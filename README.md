# meme-glossary

<h2> Install </h2> (python3 is required)

<h4>Client only usage : </h4>

```
git clone https://github.com/sngjuk/meme-glossary.git
./install.sh client
```

<h4>Full usage : </h4>

```
git clone --recurse-submodules https://github.com/sngjuk/meme-glossary.git
./install.sh full
```
